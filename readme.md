# VibeSDLC

VibeSDLC là dự án quản lý quy trình phát triển phần mềm (Software Development Life Cycle).

## Mục tiêu

- Hỗ trợ lập kế hoạch, theo dõi tiến độ và quản lý các giai đoạn phát triển phần mềm.
- Tăng hiệu quả làm việc nhóm và minh bạch quy trình.

## Tính năng

- Quản lý dự án và nhiệm vụ
- Theo dõi tiến độ từng giai đoạn SDLC
- Báo cáo và phân tích hiệu suất
- Tích hợp với các công cụ phổ biến

## Cài đặt

```bash
git clone https://github.com/yourusername/VibeSDLC.git
cd VibeSDLC
# Làm theo hướng dẫn cài đặt trong docs hoặc file hướng dẫn riêng
```

## Sử dụng

- Xem tài liệu hướng dẫn sử dụng tại [docs](./docs).
- Chạy ứng dụng theo hướng dẫn trong file `INSTALL.md`.
